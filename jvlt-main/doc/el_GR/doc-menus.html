Το μενού Αρχείο

Αρχείο->Νέο
Δημιουργεί ένα νέο λεξικό. Καθώς το jVLT μπορεί να διαχειριστεί μόνο ένα λεξικό τη φορά, πρέπει να αποθηκεύσεις το τρέχον λεξικό.

Αρχείο->Άνοιγμα
Ανοίγει ένα παράθυρο διαλόγου που σου επιτρέπει να ανοίξεις ένα νέο λεξικό.

Αρχείο->Άνοιγμα Πρόσφατου
Ανοίγει ένα υπομενού που εμφανίζει τα προσφάτως ανοιγμένα αρχεία. Επιλέγοντας ένα αντικείμενο του μενού ισοδυναμεί με το άνοιγμα του αντίστοιχου αρχείου.

Αρχείο->Αποθήκευση
Αυτή η εντολή αποθηκεύει το λεξικό.

Αρχείο->Αποθήκευση Ως
Αποθηκεύει ένα λεξικό επιτρέποντάς σου να καθορίσεις την τοποθεσία αποθήκευσης και το όνομά του.

Αρχείο->Εκτύπωση, Αρχείο->Προεπισκόπηση εκτύπωσης
Η εκτύπωση περιγράφεται εδώ.

Αρχείο->Εισαγωγή
Ανοίγει ένα παράθυρο διαλόγου που σου επιτρέπει να εισάγεις ένα λεξικό είτε σε μορφή jvlt ή σε μορφή CSV (comma separated values - οι εγγραφές του λεξικού είναι διαχωρισμένες με κόμματα).

Αρχείο->Εξαγωγή
Ανοίγει ένα παράθυρο διαλόγου που σου επιτρέπει να εξάγεις λέξεις και παραδείγματα σε αρχείο μορφής jvlt ή CSV.

Αρχείο->Έξοδος
Τερματισμός της εφαρμογής. Εάν το τρέχον λεξικό έχει τροποποιηθεί θα ερωτηθείς εάν θέλεις να αποθηκεύσεις τις αλλαγές ή όχι.

Το μενού Επεξεργασία

Επεξεργασία->Αναίρεση
Αναίρεση της τελευταίας εντολής που τροποποίησε το περιεχόμενο του λεξικού. Τέτοιες εντολές μπορεί να είναι η προσθήκη, η επεξεργασία, η αφαίρεση λέξης ή η εφαρμογή των αποτελεσμάτων ενός κουίζ.

Επεξεργασία->Επανάληψη
Επανάληψη του τελευταίου βήματος ακύρωσης.

Επεξεργασία->Ιδιότητες
Επεξεργασία των ιδιοτήτων του λεξικού. Προς το παρόν η μόνη ιδιότητα που υπάρχει είναι η γλώσσα. Ο καθορισμός γλώσσας είναι προαιρετικός και επηρεάζει μόνο τα προσαρμοσμένα πεδία όπως το φύλο ενός ουσιαστικό, την συζυγία του ρήματος κλπ. τα οποία μπορείς να θέσεις στο παράθυρο διαλόγου για προχωρημένες ρυθμίσεις για μία λέξη.

Το μενού Εργαλεία

Εργαλεία->Επαναφορά στατιστικών
Αυτή η επιλογή του μενού ανοίγει νέο παράθυρο διαλόγου που σου επιτρέπει να απομακρύνεις τα στατιστικά (αριθμός σφαλμάτων κλπ) για όλες τις λέξεις ή μόνο για όσες λέξεις εμφανίζονται στη λίστα λέξεων (μπορείς να χρησιμοποιήσεις φίλτρο για να μειώσεις τον αριθμό των λέξεων που εμφανίζονται στη λίστα).

Εργαλεία->Εμφάνιση αρχείου καταγραφής σφαλμάτων.
Αυτή επιλογή του μενού ανοίγει ένα νέο παράθυρο που εμφανίζει τα μηνύματα σφαλμάτων που έχουν παραχθεί από το jVLT. Εάν παρατηρήσεις κάποια μη φυσιολογική συμπεριφορά (π.χ. πάτημα ενός κουμπιού χωρίς καμία αντίδραση) μπορείς να συμβουλευτείς το αρχείο καταγραφής σφαλμάτων και να ενημερώσεις τους συγγραφείς.

Εργαλεία->Ρυθμίσεις
Αυτή η εντολή σου επιτρέπει να επεξεργαστείς ορισμένες ρυθμίσεις.

Το μενού Βοήθεια

Βοήθεια->jVLT Βοήθεια
Αυτή η εντολή ανοίγει ένα νέο παράθυρο που εμφανίζει τις οδηγίες χρήσης του jVLT.

Βοήθεια->Σχετικά με το jVLT
Αυτή η επιλογή εμφανίζει πληροφορίες σχετικά με την έκδοση και τους συγγραφείς.
